# Assignment Submission

## 📌 Submission Instructions

Please complete the following steps before the deadline:

### 1. Upload Assignment to GitHub
- Push your complete project/code to a GitHub repository.
- Ensure the repository is public or accessible.

### 2. Create a Demo Video
- Record a short demo video explaining:
  - Project overview
  - Features/functionality
  - How to run the project
- Upload the video to:
  - Google Drive **OR**
  - GitHub
- Make sure the video link is public.

### 3. Submit the Assignment
Submit the following through the Google Form:
- GitHub Repository Link
- Demo Video Link
