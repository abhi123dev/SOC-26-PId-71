## 📌 Submission Instructions

Please complete the following steps before the deadline:

### 1. Upload Assignment to GitHub
- Push your complete project/code to a GitHub repository.
- Ensure the repository is public or accessible.

### 2. Create a Demo Video
- Record a short demo video explaining:
  - Demo the Final Gameplay
    - Show:
      - Select number
      - Roll dice
      - Correct guess (score increases)
      - Wrong guess (score decreases)
      - Reset score
      - Rules toggle
      - Error when no number is selected

  - Explain the Logic (Most Important Section)
    - Explain the game flow
    - Explain component structure and why you created those component
    - How dice values are generated
    - How score is calculated
    - How error handling works

  - Explain what React Concepts you used and why they are needed

  - Challenges Faced
    - Mention:
      - Atleast one challenge you encountered
      - How you solved it

- 📌 Important 
     - Your explanation should focus on:
          - How the application works
         - Why you made certain decisions
          - React concepts used

- Show how to run the project from scratch

Not just a UI walkthrough.

- Upload the video to:
  - Google Drive **OR**
  - GitHub
- Make sure the video link is public.

### 3. Submit the Assignment
Submit the following through the Google Form:
- GitHub Repository Link
- Demo Video Link
